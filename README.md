# Gene Expression Cancer Classification using Deep Learning

## Overview
This project uses Deep Learning to classify cancer types from gene expression RNA-Seq data.

## Dataset
- Source: Kaggle
- Samples: 801
- Features (Genes): 20,531
- Multiple cancer classes

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- Matplotlib
- Seaborn

## Methodology
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. Train-Test Split
5. Feature Scaling
6. Deep Neural Network Model
7. Model Evaluation

## Model Architecture
- Dense (128, ReLU)
- Dropout (0.3)
- Dense (64, ReLU)
- Dropout (0.3)
- Output Layer (Softmax)

## Results
- Training Accuracy: 99.6%
- Validation Accuracy: 99.2%
- Test Accuracy: 100%
